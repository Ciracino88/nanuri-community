## 먼저 읽을 것

**지금 [원티드 디자인 시스템 전면 개편](status.md#-지금-진행-중-원티드-디자인-시스템-전면-개편)이
진행 중이라 앱이 어중간한 상태입니다.**

두 가지만 미리 알아두세요.

- **탭바가 없습니다.** 옛것을 걷어내고 새것(떠 있는 글래스 캡슐)은 아직 안 붙였습니다.
  로그인하면 `/home`에 떨어지고 거기 빠른 메뉴 말고는 이동할 방법이 없습니다. 버그가 아닙니다.
- **화면 대부분이 아직 옛 디자인입니다.** 소모임 하나만 넘어갔습니다. 흰 배경에 안 보이는
  글자나 뒤집힌 색이 보이면 폐기된 다크 재디자인의 잔재입니다.

화면을 손대기 전에 [design.md](design.md)와 [status.md](status.md) 상단을 보세요.

> 앞서 있던 "라이트 리디자인 3단계"는 이 작업에 흡수되어 **폐기**됐습니다. 옛 문서나 커밋에서
> 카테고리 틴트(teal/pink/amber)·액센트 퍼플을 보더라도 지금은 없는 개념입니다.
